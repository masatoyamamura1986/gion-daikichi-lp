# PRD: 祇園だいきち牧場 LP

## 概要

祇園だいきち牧場のランディングページを、Astro + Cloudflare Pages + microCMS で構築する。
現在の静的 HTML（日本語版・英語版）を Astro に移行し、テキスト・画像を microCMS で管理可能にする。

## 背景・目的

- 現状: 静的 HTML で日本語版（`index.html`）と英語版（`en/index.html`）を運用
- 課題: テキスト・画像の更新にコード編集が必要
- 目的: 非エンジニアでも microCMS 管理画面からコンテンツを更新できるようにする

## ターゲットユーザー

- **閲覧者**: 祇園エリアで食事を探す国内外の観光客・地元客
- **管理者**: 店舗スタッフ（microCMS でコンテンツを更新）

## 機能要件

### F1: 多言語対応（日本語 / 英語）

- `/` で日本語版、`/en/` で英語版を表示
- 言語切替リンク（JP / EN）をヘッダーに配置
- microCMS 上で日本語・英語テキストを同一コンテンツ内で管理

### F2: ヒーローセクション

- スライドショー画像（複数枚、自動切替 5秒間隔）
- ロゴオーバーレイ
- スクロールインジケーター

### F3: キャッチコピーセクション

- メインコピー（日/英）
- サブコピー（日/英）

### F4: About セクション

- 外観画像
- セクション見出し（日/英）
- 説明文（日/英）
- 外部リンクボタン 2つ（ラベル日/英 + URL）

### F5: メニューセクション

- **おすすめ料理**（リスト表示）
  - 料理画像、料理名（日/英）、価格、説明文（日/英）、動画リンク
- **コラボメニュー**（リスト表示）
  - 同上の構造
- その他メニューリンク（PDF 等）

### F6: 店舗情報セクション

- Google Map 埋め込み
- 内観画像
- 住所（日/英）、TEL、営業時間、定休日（日/英）、お支払い（日/英）、予約情報（日/英）
- 系列店舗リンク

### F7: フッター

- ページ内ナビ
- ロゴ
- 住所（日/英）、TEL
- SNS リンク（Instagram, TikTok, YouTube）

### F8: ナビゲーション

- ハンバーガーメニュー → スライドパネル
- ページ内アンカーリンク（スムーズスクロール）
- SNS リンク

## 非機能要件

### パフォーマンス

- Lighthouse Performance スコア 90 以上
- SSG（静的サイト生成）で配信
- 画像は microCMS の画像 API またはローカル最適化を利用

### SEO

- 各言語ごとに適切な meta タグ（title, description, OG, Twitter Card）
- `hreflang` による多言語リンク
- canonical URL 設定

### デプロイ・運用

- Cloudflare Pages でホスティング
- microCMS Webhook → Cloudflare Pages ビルドトリガーで自動デプロイ
- ビルド時間: 1分以内目標

### ブラウザ対応

- Chrome, Safari, Firefox, Edge（最新2バージョン）
- iOS Safari, Android Chrome

### レスポンシブ

- PC（1280px）、SP（768px 以下）の 2ブレークポイント
- 現行デザインを踏襲

## ページ構成

| URL | 内容 |
|---|---|
| `/` | 日本語版 LP |
| `/en/` | 英語版 LP |

## 補足

- 本プロジェクトでは DB を使用しないため `DB_SCHEMA.md` は不要
- SSG（静的サイト生成）のため Cloudflare アダプタ（SSR用）は不要
